Unix, Git, and tooling basics for students taking digital design and verification courses.

## Quick start

**Module 1** (Linux & Shell Basics):

```bash
./scripts/module1.sh --check   # self-check
./scripts/module1.sh --demo    # print demo commands
./scripts/module1.sh --scaffold # create ~/unix_practice for exercises
```

**Module 2** (Filesystem, Permissions & Environment):

```bash
./scripts/module2.sh --check   # self-check
./scripts/module2.sh --demo    # print demo commands
./scripts/module2.sh --scaffold # copy hello.sh to ~/unix_practice for chmod practice
```

**Module 3** (Processes, Pipes & Redirection):

```bash
./scripts/module3.sh --check   # self-check
./scripts/module3.sh --demo    # print demo commands
./scripts/module3.sh --scaffold # copy sample.log to ~/unix_practice for grep practice
```
